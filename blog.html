Question-1: What are the different ways to select an element in the DOM?

The different ways to select an element in the DOM includes getElementById() method, getElementsByTagName() method, querySelector() method, querySelectorAll() and getElementByClassName() methods.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The innerHTML property reads the html elements as well as the text within them, whereas the innerText property ignores the html elements and returns only the text which is displayed in browser, the textContent property also shows only the text within the html tags but it also shows any hidden text that is not rendered in the browser.

Question-3: What is event delegation in the DOM?

Event delegation is a smart way to handle events. Instead of adding eventlistener to every child elements , a single eventlister can be added to the parent element. It works by tracking the target element by using .target property of the event object and triggering the eventlistener when the target element is clicked or some other predefined action is done with it.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process where an event is triggered in a child element and that triggers the ancestor elements up the DOM tree as well. So the event propagates upward from the child to the parent elements.

Question-5: How do you create, add, and remove elements using JavaScript?

New elements are created using the createElement() method, newly created elements are added to parent element using appendChild() method.The removeChild() method can be used to remove any child element from the parent node.
